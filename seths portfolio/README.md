# Seth Cryptolord Portfolio

A modern, terminal-style portfolio website for Seth Cryptolord - CEO & Founder of MetaHub Academy and BullBear Trading.

## About Seth

Seth Cryptolord is a digital entrepreneur, visionary educator, and Web3 enthusiast with over 10 years of experience in forex, crypto trading, and digital business. He founded MetaHub Academy in 2017, which has grown from Migori to Nakuru to Nairobi, empowering thousands across Africa to achieve financial freedom.

## Features

- **Interactive Terminal Interface**: Command-line style portfolio with real-time commands
- **Responsive Design**: Works perfectly on desktop, tablet, and mobile devices
- **Dark/Light Mode**: Toggle between themes with `light` and `dark` commands
- **AI Assistant**: Interactive AI helper to answer questions about Seth
- **3D Profile Card**: Interactive card with hover effects
- **Live Date/Time**: Real-time clock display
- **Smooth Animations**: Professional typewriter effects and transitions

## Available Commands

Type these commands in the terminal:

- `help` - Show all available commands
- `about` - Learn about Seth Cryptolord
- `bio` - Short biography
- `skills` - View technical skills and expertise
- `experience` - Work experience and ventures
- `education` - Educational background
- `projects` - Featured projects (MetaHub Academy, BullBear Trading, etc.)
- `trading` - Trading expertise in forex and crypto
- `services` - Services offered
- `contact` - Get contact information
- `email` - Get email address
- `social` - View social media links
- `quote` - Get a random inspirational quote
- `ai [question]` - Ask the AI assistant anything
- `sudo` - Execute with elevated privileges
- `light` - Switch to light mode
- `dark` - Switch to dark mode
- `clear` - Clear terminal

## Setup Instructions

1. **Add Profile Image**: 
   - Place Seth's profile image in the `images` folder
   - Name it `seth.jpeg` (or update the HTML if using a different name)

2. **Open the Portfolio**:
   - Simply open `index.html` in your web browser
   - Or use a local server for best experience

3. **Customize**:
   - Update contact information in `script.js`
   - Modify colors in `styles.css` (Bitcoin orange theme: #f7931a)
   - Add more commands or content as needed
